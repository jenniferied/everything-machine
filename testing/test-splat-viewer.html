About This Test Page

This is a standalone test page for the GaussianSplatViewer component using gsplat.js.

Gaussian Splatting is a novel 3D representation technique that uses millions of 3D Gaussians to render photorealistic scenes in real-time.

Features:

  • Real-time rendering of 3D Gaussian Splats
  • OrbitControls for camera navigation
  • WebGL2 required for rendering
  • Demo model: Nike Shoe from Hugging Face
