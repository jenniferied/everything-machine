About This Test Page

This is a standalone test page for the ThreeDViewer component. It loads Three.js and displays 3D models with PBR materials.

Features:

  • Multiple demo models (Damaged Helmet, Horse, Flamingo, etc.)
  • Lighting modes: Studio, Daylight, Night, HDRI
  • Texture isolation (Diffuse, Normal, Roughness, etc.)
  • OrbitControls for camera interaction
  • Wireframe toggle
